
## 🚀 기능

### 초기 설정
- [x] 지하철역으로 교대 강남 역삼 남부터미널 양재 양재시민의숲 
  매봉역이 등록되어져 있다.
- [x] 노선으로 2호선 3호선 신분당선이 등록되어져 있다.
  - [x] 2호선 : 교대역 - 강남역 - 역삼역
  - [x] 3호선: 교대역 - 남부터미널역 - 양재역 - 매봉역
  - [x] 신분당선: 강남역 - 양재역 - 양재시민의숲역

### 구현해야할 기능
- [ ] 역 관리
    - [ ] 역 등록하기 기능
      - [ ] 중복된 이름의 역은 등록 불가능
      - [ ] 지하철 역의 이름은 두 글자 이상이여야 한다.
    - [ ] 역 삭제하기 기능
      - [ ] 노선에 등록된 역들은 삭제 불가능
    - [ ] 역 조회하기 기능
    - [ ] 돌아가기
- [ ] 노선 관리 
    - [ ] 노선 등록하기 기능
      - [ ] 중복된 이름의 노선은 등록 불가능
      - [ ] 노선의 이름은 두 글자 이상
      - [ ] 상행 종점역과 하행 종점역을 입력받는다.
    - [ ] 노선 삭제하기 기능
    - [ ] 노선 조회하기 기능
    - [ ] 돌아가기
- [ ] 구간 관리
  - [ ] 구간 등록하기 기능
  - [ ] 구간 삭제하기 기능
    - [ ] 노선엔 포함된 역이 두 개 이하일 때는 역 제거할 수 없다.
    - [ ] 종점 제거 시 종점 이전 역이 종점이 된다.
  - [ ] 돌아가기
- [ ] 지하철 노선도 출력
- [ ] 종료
